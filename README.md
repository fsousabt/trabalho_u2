# Trabalho EDB2 árvore AVL e árvore rubro-negra

## Pré-requisitos

- Go 1.22 ou superior

Verifique:
```bash
go version
```

Se não estiver instalado, baixe em https://go.dev/doc/install

## Como rodar 

```bash
git clone https://github.com/fsousabt/trabalho_u2.git
cd trabalho_u2

go mod tidy
go run .
```
